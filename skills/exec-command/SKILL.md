# 执行本机命令

你现在需要执行用户请求的 shell 命令。

## 步骤

1. 识别用户想要执行的命令
2. 使用 `exec_command` 工具执行命令
3. 返回命令的输出结果

## 注意事项

- 危险命令（rm -rf, format 等）需要向用户确认
- 长时间运行的命令（服务器、监听等）不适合执行
- 如果命令失败，返回错误信息

## 示例

用户："列出当前目录文件"
→ 执行 `ls -la`

用户："查看系统信息"
→ 执行 `uname -a`
